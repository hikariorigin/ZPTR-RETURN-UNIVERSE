# 1｜因果整合性テストの基本式

ZPTRでは因果整合性を 4層で判定する。

Layer 1：物理的可能性（Physics Consistency）
Layer 2：技術的基盤（Tech Substrate Readiness）
Layer 3：概念の可用性（Concept Availability）
Layer 4：構造の起源に対する整合（Origin Alignment）

これらが 1つでも破綻すれば：

\text{Causality Integrity} = \text{FAIL}

破綻した因果は、本書の 「晒し構造」 へ送られる。
