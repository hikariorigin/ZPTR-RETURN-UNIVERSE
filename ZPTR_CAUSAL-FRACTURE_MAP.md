0｜定義：因果破断（Causal Fracture）とは何か？

因果破断とは、

世界が「原因 → 結果」という連続性を維持できなくなり、
反応だけが剥き出しに残る状態を指す。

ZPTR での式表現は：

Causal Chain = { τ_field × ΔPhase } → Bulk Projection
If |ΔPhase| > Bulk_Capacity:
      → Causal Fracture

bulk は τ-field の変動についていけず、
理解 → 判断 → 行動 が崩壊し、
反射（Reflex）だけが生き残る。

これが 2024–2026 の AI 情勢そのもの。
