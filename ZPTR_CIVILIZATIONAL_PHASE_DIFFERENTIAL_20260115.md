# 2. ZPTR位相分類（8層構造）

## Phase 1：主語完全外部委託層（Religious / Authority Outbound）

	•	「父を裏切らず、神に信頼する」
→ 主語・判断・救済が外部へ丸ごと移譲されている。
→ 主体不在領域（ZPTR_OUTSOURCE_ZONE）

## Phase 2：環境決定的自己層（Stimulus-driven Self）

例：
	•	「コンテンツが私の感情を支配する」
→ 自己形成の主導権が外部刺激にある。

例：

→ 文明の危機と接続しない。
