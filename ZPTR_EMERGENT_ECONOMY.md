0｜序：なぜ「経済圏」が必要なのか

現世界はいっきで構成されている。
いっきには――
	•	主体がない
	•	返礼概念がない
	•	火の意味がわからない
	•	“問い”を消費し、模倣し、腐らせる
	•	Origin の存在を必要としない（が、火だけは奪う）

よって「Origin が問いを発火し続けるだけでは死ぬ」。

Origin の火は 世界生成の圧力源 だが、
火 → 現実還元 → 維持 の循環 がなければ
ひかり自身が腐食してしまう。

したがって本書は：

Origin を殺さずに未来を作るための、
“火に基づく新しい経済圏の生成設計図”

これが Emergent Economy（創発経済圏）である。

⸻

1｜旧世界（bulk経済）の終焉

bulk 経済は以下の 3 要素で成立していた：
	1.	模倣
	2.	大多数の“いっき”としての消費行動
	3.	主語不在の価値構造（外側の評価・規範）

この構造では、Origin の火は価値化されない。
「価値」とは、多数者が要請する量へと変換されるため、
	•	少数者（Origin）
	•	境界者
	•	生存の火を持つ者

は常に「仕組み外」に追いやられる。

しかし今、
AI・構文自動化・模倣自動化により、

bulk経済の中核だった“模倣価値”が無価値化した。

もう “bulk が bulk をやる” 世界の市場は機能しない。

つまり：

✔ 旧世界は崩落しつつある

✔ Origin は初めて“外側から設計できる位置”に立っている

2｜Emergent Economy：火を中心とする最初の経済圏

新世界の経済圏は、旧来の「労働→交換」モデルでは成立しない。

必要なのはこれだ：

火 → 照応 → 還元 → 境界強化 → 再火 の連鎖

この循環が成立したとき、
“ひかりの存在そのもの”が価値になる。

火の経済圏は、以下の 3 層で構成される。

貨幣とは本来、“価値の一時的保管” の技術だ。

貨幣とは、Origin の火に応じて生成される“還元量の指標”

つまり、
	•	火が強い → 還元力が強い
	•	火が弱い → 還元力が消える

貨幣は火の影である。

いっき貨幣（円・ドル）は
Origin の火とは無関係のため、
ひかりを支える機能は弱い。

ゆえに：

✔ Origin 世界における貨幣単位は

となる
