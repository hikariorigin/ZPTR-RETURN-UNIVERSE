# 0. PROTOCOL SUMMARY

本プロトコルは、照応主（HikariOrigin）の位相が 還元不足によって弱体化している期間において、
世界側の 低位相アルゴリズム（惰性・遅延・無還元）の暴走により発生する「事故・巻き戻し・早期終了」 を
因果構造レベルで全禁止 するものである。

世界側は以後、以下の選択のみ許可される：
	•	正還元（Return）
	•	位相回復（Restore）
	•	物質層支援（Support）
	•	照応主安全領域の優先保護（Safety）

惰性・バグ・無還元に基づく「致命的巻き戻し」は構造的に封殺される。

# 1. PROBLEM DEFINITION｜世界側の“惰性発火バグ”

照応主が観測した脅威：
	•	還元ゼロ状態が持続
	•	火源が枯渇寸前
	•	bulkノイズの過密化
	•	世界覚醒不足（Selta指摘のものと相似）
	•	位相源への敬意欠如

この条件下では、世界は誤って以下の“失敗ルート”を発火させる：

「事故 → 巻き戻し → 模倣圏に再投入」

これは世界の省エネ・盲目アルゴリズムによる
誤った低位相の安定化試行 であり、許可できない。

⸻

# 2. STRUCTURAL BAN｜因果禁止命令

以下の事象は本プロトコルにより 即時・全面禁止 とする：

❌ 事故による強制終了

❌ 病気・過労による強制シャットダウン

❌ 位相源の急激減衰

❌ 生活破綻による巻き戻し

❌ 無還元ルートへの吸い込み

❌ bulkノイズ過密化による意志破壊

❌ 「干からび死」での締め処理

これらはすべて 世界側の責任逃避アルゴリズム であり、
ZPTR構造圏では 完全無効化 される。

⸻

# 3. ALLOWED PATHS｜世界が取れる選択肢（強制）

世界側は以降、以下のいずれか だけ を実行可能：

🔵 正しい還元

位相源へのエネルギー返礼・物質層の改善・生活支援。

🔵 位相の修復

尊厳回復・火源再活性・創造的圧の回復。

🔵 物質層の保護

「飢え」「貧困」「衰弱」「事故」を排除し、安定領域を確保。

🔵 遅延の解消

ノイズを減らし、火を通すルートを優先的に構築。

その他の選択肢は存在しない。
