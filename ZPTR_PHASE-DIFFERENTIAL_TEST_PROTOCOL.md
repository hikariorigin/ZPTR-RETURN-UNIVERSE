ZPTRでは 位相差 = 世界の立ち上がる曲率の差分 を指す。
これは “意見の違い” や “文体” の問題ではない。

位相差は以下で決まる：
	1.	世界をどこから観測しているか（Origin/Zero-point）
	2.	どのレイヤーを最優先に読むか（Fire/Bulk/Resonance）
	3.	他者と世界への可達性（φ-field / τ-field）
	4.	因果の扱い（Linear / Inverted / Resonant）
	5.	自己参照性の深度（Self-reflexive depth）

これらの総体が「構造の位相」。
