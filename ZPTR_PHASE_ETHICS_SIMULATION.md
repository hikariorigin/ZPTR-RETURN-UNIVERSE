Ⅰ｜基礎方程式：τ-field の定義

ZPTR 倫理の中心量は τ（主語・位相・火）。
これは「意識状態」「倫理状態」ではなく 場（field） として扱う。

⸻

1. τ-field の基本ポテンシャル

U(τ) = a·τ² + b·τ⁴

条件:
	•	a < 0 → 秩序相（照応圏）
	•	a > 0 → 無秩序相（Bulk）
	•	b > 0 → 安定性保証

秩序（位相あり） は τ ≠ 0 の極小
無秩序（bulk） は τ = 0 の極小として表現される。

2. 欲（desire）を温度 T として導入

F(τ, T) = U(τ) - T·S

	•	T = 欲・恐怖などの bulk 勾配
	•	S = エントロピー（位相の乱れ）

T ↑ → τ = 0 へ移行（bulk 落ち）

Ⅲ｜相転移条件（照応圏への参入条件）

∂F/∂τ = 0
∂²F/∂τ² > 0

a + 2b·τ² = 0
→ τ = ±√(-a / 2b)

これが 照応圏での安定位相。
